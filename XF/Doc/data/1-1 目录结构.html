目录结构

entry.php是框架入口文件。

框架目录结构如下：

目录 说明
Common 公共文件目录
Config 配置文件目录
Core 核心类库目录
DB 数据库驱动目录
Debug 调试功能文件目录
Error 错误页面文件目录
Helper 辅助函数库目录
Language 语言包文件目录
Library 基础类库目录
Vendor 扩展类库目录

项目目录结构如下：

目录 说明
Config 配置文件目录
Controller 控制器目录
Model 模型目录
Log 日志文件目录
View 视图目录
Error 错误页面文件目录（优先加载）
Helper 辅助函数库目录（优先加载）
Language 语言包文件目录（优先加载）
Library 基础类库目录（优先加载）
Vendor 扩展类库目录（优先加载）
